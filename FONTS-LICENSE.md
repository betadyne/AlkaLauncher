## SIL Open Font License 1.1 Summary

The SIL Open Font License (OFL) is a free, open-source license specifically designed for fonts. It allows you to:

✅ **Use** the fonts freely in any project (personal, commercial, etc.)
✅ **Bundle** the fonts with software or applications
✅ **Distribute** the fonts with your application
✅ **Modify** the fonts (but modified versions must also be under OFL)

The only restriction is that the fonts cannot be sold by themselves.

For the full license text, see: https://openfontlicense.org/open-font-license-official-text/
